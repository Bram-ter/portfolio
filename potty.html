Potty made with JavaScript

An interactive tamagotchi made with HTML, CSS and JavaScript. While focusing primarily on the JavaScript bit.

When I started working on this project I had little to no experience with JavaScript. On my previous education (Grafisch Lyceum Utrecht) I learned how to make basic if and else statements, and that was it. So I expected it to be a big challenge making an interactive tamagotchi.

Lucky for me, my teacher had a lot of examples I could refresh and learn from. The great thing about these examples were that they were really basic, to the point you could add code to them yourself really easily and make it your own.

For example, take the image of the code. The default code shown here was for a slider. After a lot of time tinkering, I made it so that you could “evolve” the teapot in different elements. If you would hover a certain stone over the teapot, JavaScript would check which value the stone had, and then it would change the body to that element by making the value of the body the same as that stone you just hovered with.
