FED website

A project where I chose an existing website, and had to try make a copy of it while learning more about frontend of websites.

While I already knew some things about frontend because of my internship, I found this course incredibly helpful. My internship was already a couple of years ago, so there were some things that got updated without me noticing. And since I was not a pro there was a lot of room for further improvement.

In this course I learned a lot of new things:

  • I learned how to animate elements with keyframes. For the animations that played when a section would come into the viewport of the user, there were short slide-out animations. I’ve never really used keyframes before so this was very interesting to experiment with.
  • I learned how the intersection observer works with JavaScript. This API came in handy for when a section would come into the viewport. This made it possible for keyframes to only start playing when a user would scroll to a section. Cause you wouldn’t want all the animations to play at once if the user would visit the page.
  • I learned how to position elements with CSS grid and flexbox. I was already a bit familiar with flexbox, but didn’t knew all the various things I could do with it. I used the lessons to refresh, and also learn a new way to position elements with CSS grid.

When I started working on the project, I first had to get a good grip on all the animations and interactable elements in the website. To get a better grip I made an analysis in Adobe XD, where I made screenshots of the website and surrounded the elements with a square. I also added some annotations to indicate what elements something was made out of.
